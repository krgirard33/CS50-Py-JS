Hello, Git!

Notes for Powershell commandline

  • New-Item file.name -- creates new file
  • cd - change directory
  • ls - list everything in directory
  • git add file.name - Tells git to keep an eye on this file
  • git commit -m "message" - saves a snapshot of the file
  • git commit -am "message" - saves a snapshot of all changed/added files in folder/directory
  • git status - tells you what is happening in local depository
  • git push - pushes commited changes to github
  • git restore file.name - to discard changes in local directory
  • git pull - pull down the changes from github to my local repository
  • merge conflict - my changes are between <<< head & ====. The remote conflict is between === & >>>
  • git log - give you a list of all commits
  • git reset --hard <commit you want to go back to> -- takes you back to that commit
  • git reset --hard origin/master -- reverts you back to what is on the origin/master branch
  • git branch - tells you what branches you have
  • git checkout -b branch.name ---- makes a new branch
  • git checkout branch.name ---- move you to an existing branch
  • git merge branch.name -- merges branch.name branch with main/master branch
  • After fixing a merge conflict, commit the file
